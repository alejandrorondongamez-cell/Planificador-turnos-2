# Aequitas WFM

Plataforma de alta fidelidad para el control y balance óptimo de turnos IT con sede en Alicante.

## Configuración de Producción
1. Crea los archivos en la rama principal (`main`).
2. Ve a los **Settings** del repositorio.
3. En la pestaña **Pages**, selecciona la rama `main` y la carpeta raíz (`/`) como origen.
4. Salva la configuración y tendrás la URL pública HTTPS de inmediato.

*Contraseña Administrador por defecto:* `admin123`
